MacOS Install Instructions

Step 1: Get the right device

Before you go any further, make sure you have one of these devices.

Step 2: Download CalyxOS

Download the corresponsing file for the type of device that you have.

Step 3: Download the flasher

You'll use this to get the operating system onto your device.

Download

Step 4: Prepare the files

Place the flasher as well as the CalyxOS image into the same folder. Do not extract or rename the zip, simply copy it as-is.

Step 5: Install CalyxOS

Open a Terminal or Command Prompt in that directory, and then Run the flasher using:

chmod +x ./device-flasher.darwin; ./device-flasher.darwin
