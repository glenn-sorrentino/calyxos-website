Features

Learn about some of the ways we protect your privacy.

Private by Design

Lock Down your Data

👍 Storage encryption is on by default

🔑 Yubikey two-factor authentication support.

🦾 Built-in encrypted backup and restore suite 'SeedVault' for online (using NextCloud) or off-line backups (using USB storage) of the phone's data and apps

Security Out-of-the-Box

🔐 Full Verified Boot ensures that the operating system has not been modified, bootloader re-locking provides physical security equivalent to stock Android

🙈 Scramble lockscreen PIN

🛑 Block unknown USB devices

Don't Panic

🚨 Panic button functionality lets a user uninstall apps, remove work profile, or factory reset in an emergency.

Trust & Transparency

⚙️ Privacy settings lets you see what apps are requesting which permissions

📆 Security updates for the Operating System every month via automatic over-the-air updates

Proactive Protection

⏲ Bluetooth timeout lets you have bluetooth turn off after user-definable period of inactivity (when no devices connected)

⏰ Wifi timeout lets you have Wifi turn off after user-definable period of inactivity (when not connected to any network)

🔎 All builds / updates are tested

Customized Connections

🤝 Tethered network devices (USB or Wifi) can use the phone's VPN or Tor

🚦 Built-in Firewall 'Datura' lets you control network access for each app

🚧 Restrict cleartext traffic completely, globally.

Layers of Control

🌎 Cloudflare DNS is available as the default Private DNS provider

📞 eSim support and dual-sim support, where supported by the hardware

Curated Apps

🎁 Choose what to install from a list of trusted free software applications.

🦆 DuckDuckGo Browser is the default web browser and search engine, which blocks ads and trackers

🛍 Use the F-Droid and Aurora stores to install from the Google Play catalog anonymously

🥸 Ships with Tor Browser, Orbot, and multiple free and trusted non-logging VPNs.

Safe Communication

📧 OpenPGP encrypted email support using hardware tokens is built in using OpenKeychain and K-9 Email client

💬 Built in support for Signal and WhatsApp calling, and phone ships with Signal

🤫 Sensitive Numbers privacy. Calls to numbers for help lines such as domestic violence, child abuse, suicide hotlines are not recorded in the call log

📇 Access these numbers in the Helplines contact entry

App Compatability

✌️ microG - Available, but completely optional, replaces some functions of Google Play Services while maintaining much more anonymity and privacy.

🗺️ Mozilla Location Services (and Dejavu) available as default location services.

📍 Nominatim available as default geocoding service.
